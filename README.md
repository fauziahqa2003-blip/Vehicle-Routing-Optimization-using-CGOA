## Overview

This repository contains my undergraduate thesis project entitled:

"Implementation of the Chaotic Grasshopper Optimization Algorithm (CGOA) for Solving the Vehicle Routing Problem with Simultaneous Pickup and Delivery (VRPSPD)."

The project focuses on optimizing delivery routes by minimizing total travel distance while satisfying vehicle capacity constraints.

## Features

- Implementation of CGOA
- Supports Small, Medium, and Large datasets
- Route optimization
- Fitness evaluation
- Vehicle capacity constraints

## Algorithm

Chaotic Grasshopper Optimization Algorithm (CGOA)
Problem: Vehicle Routing Problem with Simultaneous Pickup and Delivery (VRPSPD)
Programming Language: C++
IDE: Visual Studio Code

## Dataset

Three benchmark datasets were used.

| Dataset  | Customers |
|----------|-----------|
| Small    |     13    |
| Medium   |     22    |
| Large    |    100    |

## Experimental Results

| Dataset  | Best Distance |
|----------|---------------|
| Small    |       94      |
| Medium   |      134      |
| Large    |    2737.03    |
